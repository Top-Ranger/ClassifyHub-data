ClassifyHub-data
   by Marcus Soll and Malte Vosgerau

-------------------------------------------------------------------------------

This repository contains 681 classified Repositories belonging to the following
classes:

 * DEV: software development projects
 * HW: Solutions for homeworks, exercises and similar
 * EDU: Repositories for didactic content
 * DOCS: Non-didactic documents
 * WEB: (Personal) web pages
 * DATA: Datasets
 * OTHER: Anything not contained in the above categories
 
Although these were created especially for the ClassifyHub application
(https://github.com/Top-Ranger/ClassifyHub) you can use them for any purpose
not violating the license.
